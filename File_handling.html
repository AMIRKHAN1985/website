2. File handing:¶

File is a named location on disk to store related information. It is used to permanently store data in a non-volatile memory (e.g. hard disk).

Since, random access memory (RAM) is volatile which loses its data when computer is turned off, we use files for future use of the data.

Hence, in Python, a file operation takes place in the following order.

  1. Open a file
  2. Read or write (perform operation)
  3. Close the file

a).How to open a file¶

Python has a built-in function open() to open a file. This function returns a file object, also called a handle, as it is used to read or modify the file accordingly.

In [ ]:
f = open("test.txt")      # equivalent to 'r' or 'rt'
f = open("test.txt",'w')  # write in text mode
f = open("img.bmp",'rb') # read and write in binary mode

d).How to write to File¶

In order to write into a file in Python, we need to open it in write 'w', append 'a' or exclusive creation 'x' mode.

1.Working of write mode:

It will create a file if file is not present and write content on it and if file is present it will remove all previous content and write new one.

In [ ]:
f = open("test.txt",'w')
f.write("my first file\n")
f.write("This file\n\n")
f.write("contains three lines\n")
f.close()

2.Working of append mode :

It will create a file if file is not present and write content on it and if file is present it will update file by adding content to end of file.

In [ ]:
file = open('text.txt','a') 
file.write("This will add this line") 
file.close()

3.Working of exclusive creation mode:

It is used to create new file.

In [ ]:
file = open('text1.txt','x')  
file.close()

File handling methods:¶

a).read()¶

The read() method returns the specified number of bytes from the file. Default is -1 which means the whole file.

In [ ]:
f = open("file.txt", "r")
print(f.read())
f.close()
In [ ]:
f = open("file.txt", "r")
print(f.read(5))
print(f.read(5))
f.close()

b).readline()¶

The readline() method returns one line from the file.You can also specified how many bytes from the line to return, by using the size parameter.

In [ ]:
f = open("file.txt", "r")
print(f.readline())
print(f.readline())
f.close()
In [ ]:
f = open("file.txt", "r")
#Return only the five first bytes from the first line:
print(f.readline(5))
f.close()

c).readlines()¶

The readlines() method returns a list containing each line in the file as a list item.

If parameter is passed i will read and returns a list of the line/lines corrosponding to that bytes. If the total number of bytes returned exceeds the specified number, no more lines are returned.

In [ ]:
f = open("file.txt", "r")
print(f.readlines())
f.close()
In [ ]:
f = open("file.txt", "r")
print(f.readlines(100))
f.close()

d).seek()¶

The seek() method sets the current file position in a file stream.

In [ ]:
f = open("file.txt", "r")
f.seek(4)
print(f.read(4))
f.close()

e).tell()¶

The tell() method returns the current file position in a file stream.

In [ ]:
f = open("file.txt", "r")
print(f.tell())
In [ ]:
f = open("file.txt", "r")
print(f.readline())
print(f.tell())
f.close()

f).truncate()¶

The truncate() method resizes the file to the given number of bytes. If the size is not specified, the current position will be used.

In [ ]:
f = open("file1.txt", "a")
f.truncate(20)
f.close()

#open and read the file after the truncate:
f = open("file1.txt", "r")
print(f.read())
f.close()

g).write()¶

The write() method writes a specified text to the file.

In [ ]:
f = open("file.txt", "a")
f.write("\nHello python!")
f.close()
In [ ]:
f = open("file.txt", "r")
print(f.read())
f.close()

h).writelines()¶

The writelines() method writes the items of a list to the file.

In [ ]:
f = open("file.txt", "a")
f.writelines(["\nhow are you!", "\nOver and out."])
f.close()
In [ ]:
#open and read the file after the appending:
f = open("file.txt", "r")
print(f.read())
f.close()

i).flush()¶

The flush() method cleans out the internal buffer.

Python automatically flushes the files when closing them. But you may want to flush the data before closing any file.

In [ ]:
f = open("file.txt", "a")
f.write("Now the file has one more line!")
f.flush()
f.write("...and another one!")
f.close()
